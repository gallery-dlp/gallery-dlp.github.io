Quick start

After downloading gallery-dl.exe

Four steps from download to your first image gallery.

Step 01

Save to a folder

Place gallery-dl.exe in a permanent folder like C:\Tools\gallery-dl\ — avoid Downloads which may be auto-cleared.

Step 02

If the browser warns you

Click Keep → Keep anyway in Edge/Chrome. If SmartScreen appears: More info → Run anyway.

Step 03

Open PowerShell in the folder

Click the Explorer address bar, type powershell, press Enter. PowerShell opens directly in the right folder.

Step 04

Verify & run

Type gallery-dl --version to confirm it works, then paste any URL to start downloading.

FAQ

Download FAQ

Do I need admin rights to run gallery-dl.exe?
No. gallery-dl.exe is portable — no installation needed. Admin rights are only required if you want to add the folder to the system PATH for all users.
Do I need Python installed to use gallery-dl on Windows?
No. The gallery-dl.exe release bundles Python internally. You don't need to install Python separately — just download the .exe and run it.
How do I download from a private Instagram or Twitter account?
You need to pass cookies from your browser. Run gallery-dl --cookies-from-browser chrome "URL". See the full cookies and login guide.
How do I keep gallery-dl updated?
Open PowerShell in the gallery-dl.exe folder and run gallery-dl -U. It automatically replaces itself with the latest release.
