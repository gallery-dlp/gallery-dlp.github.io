Avvio rapido

Installare gallery-dl su Windows in 4 passaggi

Senza diritti di amministratore. Funziona su qualsiasi PC Windows 11 o Windows 10.

Passo 01

Scaricare gallery-dl.exe

Salva in qualsiasi cartella, es: C:\Tools\gallery-dl\ — senza installazione, completamente portatile.

Passo 02

Aprire PowerShell in quella cartella

Clicca sulla barra degli indirizzi di Esplora file, digita powershell e premi Invio.

Passo 03

Verificare l'installazione

Esegui gallery-dl --version. Se appare un numero di versione, sei pronto.

Passo 04

Scaricare la prima galleria

Incolla qualsiasi URL supportato e premi Invio. gallery-dl salva le immagini in risoluzione originale.

Domande frequenti

Domande frequenti su gallery-dl per Windows

gallery-dl è sicuro su Windows?
Sì. gallery-dl è completamente open source e ospitato su GitHub. Ogni riga di codice è verificata pubblicamente. Nessun software aggiuntivo, nessuna telemetria.
gallery-dl funziona su Windows 11?
Sì. gallery-dl.exe funziona su Windows 11 e Windows 10 (64 bit) senza installazione. Scarica il .exe, apri PowerShell ed esegui il primo comando.
Quali siti supporta gallery-dl?
gallery-dl supporta 200+ siti tra cui Instagram, Twitter/X, Pixiv, DeviantArt, Reddit, Tumblr, Danbooru, Flickr e molti altri. Vedi la lista completa su GitHub.
Ho bisogno di Python per usare gallery-dl su Windows?
No. La versione gallery-dl.exe include un runtime Python integrato. Scarica semplicemente il .exe ed esegui — nessuna installazione di Python necessaria.
Come aggiorno gallery-dl su Windows?
Apri PowerShell nella cartella gallery-dl ed esegui gallery-dl -U. Aggiornamento automatico all'ultima versione.
