gallery-dl Twitter/X en Windows — Descargar Fotos & Vídeos

gallery-dl descarga imágenes y vídeos en alta resolución desde líneas de tiempo, pestañas de medios, likes y tweets individuales de Twitter/X. La autenticación por cookie es necesaria desde las restricciones de API de Twitter.

⬇ Scarica gallery-dl.exe Guía de configuración de cookies →
Importante — las cookies expiran rápido: Las cookies de Twitter/X expiran con más frecuencia que otros sitios. Si gallery-dl deja repentinamente de descargar, simplemente vuelve a ejecutar con --cookies-from-browser chrome para actualizar la sesión.

Comandi

Comandos gallery-dl Twitter/X para Windows

Descargar todos los medios de la pestaña de medios de un usuario
gallery-dl --cookies-from-browser chrome "https://x.com/username/media"
Descargar medios de un tweet individual
gallery-dl --cookies-from-browser chrome "https://x.com/username/status/TWEET_ID"
Descargar línea de tiempo del usuario
gallery-dl --cookies-from-browser chrome "https://x.com/username"
Descargar pestaña de likes
gallery-dl --cookies-from-browser chrome "https://x.com/username/likes"
Descargar con carpeta de salida personalizada
gallery-dl --cookies-from-browser chrome -d "C:\Imagenes\Twitter" "https://x.com/username/media"
Descargar con cookies.txt (para tareas programadas)
gallery-dl --cookies "C:\Tools\gallery-dl\twitter-cookies.txt" "https://x.com/username/media"

Configuración para Twitter/X

Configuración Twitter/X recomendada para gallery-dl en Windows

config.json — sección Twitter
{ "extractor": { "twitter": { "cookies": "C:/Tools/gallery-dl/twitter-cookies.txt", "videos": true, "retweets": false, "quoted": true, "filename": "{author[name]}_{tweet_id}_{num}.{extension}", "sleep-request": 1.0 } } }
videos: true
Descargar también archivos de vídeo
retweets: false
Omitir retweets — solo publicaciones originales
quoted: true
Incluir medios de tweets citados
sleep-request: 1.0
Pausa entre peticiones para evitar rate limits

Preguntas frecuentes

gallery-dl Twitter/X — preguntas frecuentes

gallery-dl funziona con Twitter/X nel 2026?
Sí. El extractor Twitter/X de gallery-dl se mantiene activamente. Desde las restricciones de API de Twitter en 2023, el inicio de sesión mediante cookies es obligatorio. Mantén actualizado con gallery-dl -U.
gallery-dl Twitter dice «autenticación requerida» con cookies
Las cookies de Twitter/X expiran frecuentemente. Cierra sesión y vuelve a iniciarla en x.com en el navegador, cierra completamente el navegador, luego vuelve a ejecutar con --cookies-from-browser chrome.
gallery-dl descarga imágenes de Twitter en baja resolución
Por defecto, gallery-dl añade ?format=jpg&name=orig para solicitar la resolución original. Si aún obtienes imágenes en baja resolución, actualiza con gallery-dl -U.
¿Cómo descargo solo imágenes y omito vídeos?
Añade un filtro de imagen: gallery-dl --image-filter "extension in ('jpg','png','webp','gif')" --cookies-from-browser chrome "URL". O define "videos": false en la sección Twitter de config.json.
¿Puede gallery-dl descargar desde cuentas privadas/protegidas de Twitter?
Sí, si sigues la cuenta y has iniciado sesión. Pasa tus cookies del navegador — gallery-dl accederá a la línea de tiempo usando tu sesión autenticada.
