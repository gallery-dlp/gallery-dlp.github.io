Schritt-für-Schritt-Installation

gallery-dl in 5 Schritten unter Windows installieren

Funktioniert unter Windows 11 und Windows 10 (64-Bit). Kein Installationsassistent, keine Administratorrechte erforderlich.

Schritt 01

gallery-dl.exe herunterladen

Zur Download-Seite gehen und gallery-dl.exe von der offiziellen GitHub-Version herunterladen.

Schritt 02

In dedizierten Ordner speichern

Datei an dauerhaften Ort verschieben, z. B. C:\Tools\gallery-dl\. Downloads meiden — Dateien dort können automatisch gelöscht werden.

Schritt 03

PowerShell in diesem Ordner öffnen

Im Explorer die Adressleiste anklicken, powershell eingeben und Enter drücken. PowerShell öffnet sich direkt im richtigen Ordner.

Schritt 04

Funktionsfähigkeit prüfen

gallery-dl --version ausführen. Eine Versionsnummer wie 1.28.x sollte erscheinen. Falls ja, funktioniert es.

Schritt 05

Auf neueste Version aktualisieren

gallery-dl -U ausführen. Lädt die neueste Version herunter und ersetzt sich selbst. Regelmäßig wiederholen.

Optional

Konfigurationsdatei einrichten

gallery-dl.conf erstellen, um Standard-Ausgabeordner, Dateinamen-Vorlagen und Cookies einzustellen. Konfigurations-Anleitung →

Optional aber empfohlen

gallery-dl zum PATH unter Windows hinzufügen

Mit gallery-dl im PATH kann gallery-dl aus jedem Ordner in PowerShell oder CMD ausgeführt werden.

Grafische Methode (empfohlen)

  1. Win drücken und Umgebungsvariablen suchen
  2. Systemumgebungsvariablen bearbeiten klicken
  3. Unter Benutzervariablen Path → Bearbeiten wählen
  4. Neu klicken und Ordnerpfad einfügen, z. B. C:\Tools\gallery-dl
  5. OK bei allen Dialogen klicken, Terminal neu starten

PowerShell-Methode

Einmal in PowerShell ausführen — setzt PATH nur für aktuellen Benutzer, keine Administratorrechte nötig:

setx PATH "%PATH%;C:\Tools\gallery-dl"

PowerShell danach schließen und neu öffnen. Mit gallery-dl --version aus beliebigem Verzeichnis testen.

Hinweis: Falls nach PATH-Hinzufügung "gallery-dl wird nicht erkannt" erscheint, alle Terminal-Fenster schließen und neu öffnen — PATH-Änderungen gelten nur für neue Sitzungen.

Häufige Probleme

Installationsprobleme von gallery-dl unter Windows beheben

Windows SmartScreen-Warnung beim Ausführen von gallery-dl.exe
Auf Weitere Informationen → Trotzdem ausführen klicken. SmartScreen kennzeichnet neue ausführbare Dateien standardmäßig. gallery-dl.exe ist die unveränderte offizielle Open-Source-Version von GitHub. Alternativ: Rechtsklick → Eigenschaften → unten Zulassen ankreuzen → OK.
"gallery-dl wird nicht erkannt" in PowerShell
Zwei Optionen: gallery-dl direkt aus seinem Ordner ausführen oder den Ordner zum PATH hinzufügen (siehe Abschnitt oben). Nach PATH-Hinzufügung alle Terminal-Fenster schließen und neu öffnen.
"Zugriff verweigert" oder Datei vom Antivirenprogramm blockiert
Rechtsklick auf gallery-dl.exe → Eigenschaften → unten Zulassen ankreuzen wenn sichtbar → OK. Dies entfernt die Windows-Einschränkung "aus dem Internet heruntergeladen". Bei Blockierung durch Antivirenprogramm Ausnahme hinzufügen — es ist ein Fehlalarm für Open-Source-Portables.
gallery-dl lädt nichts herunter / beendet sich sofort
Die Seite benötigt wahrscheinlich eine Anmeldung. gallery-dl --cookies-from-browser chrome "URL" ausführen. Mehr Infos in der Cookie-Anleitung.
