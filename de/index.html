Die nützlichsten gallery-dl-Befehle — kopieren & einfügen

Direkt in PowerShell oder CMD ausführen. URL durch eigenen Link ersetzen.

Alle Bilder von einer URL herunterladen
gallery-dl "https://www.instagram.com/username/"
Mit Browser-Cookies herunterladen (für anmeldepflichtige Inhalte)
gallery-dl --cookies-from-browser chrome "URL"
In benutzerdefinierten Ordner mit Originalдатeinamen speichern
gallery-dl -d "C:\Bilder\%(category)s\%(uploader)s" "URL"
Nur Bilder herunterladen (Videos überspringen)
gallery-dl --image-filter "extension in ('jpg','png','webp')" "URL"

FAQ

Häufige Fragen zu gallery-dl für Windows

Ist gallery-dl sicher unter Windows?
Ja. gallery-dl ist vollständig Open Source und auf GitHub gehostet. Jede Codezeile ist öffentlich geprüft. Keine Zusatzsoftware, keine Telemetrie, keine Hintergrunddienste.
Funktioniert gallery-dl unter Windows 11?
Ja. gallery-dl.exe läuft unter Windows 11 und Windows 10 (64-Bit) ohne Installation. .exe herunterladen, PowerShell öffnen und ersten Befehl ausführen.
Welche Seiten unterstützt gallery-dl?
gallery-dl unterstützt 200+ Seiten, darunter Instagram, Twitter/X, Pixiv, DeviantArt, Reddit, Tumblr, Danbooru, Flickr, Artstation, Patreon und viele mehr. Die vollständige Liste auf GitHub.
Benötige ich Python für gallery-dl unter Windows?
Nein. Die gallery-dl.exe-Version enthält eine integrierte Python-Laufzeitumgebung. Einfach die .exe herunterladen und ausführen — keine Python-Installation nötig.
Wie aktualisiere ich gallery-dl unter Windows?
PowerShell im gallery-dl-Ordner öffnen und gallery-dl -U ausführen. Automatisches Update auf die neueste Version.
Warum meldet gallery-dl "Login erforderlich" oder "403 Forbidden"?
Viele Seiten erfordern eine Anmeldung für private oder mitgliederpflichtige Inhalte. --cookies-from-browser chrome verwenden, um die Browser-Sitzung weiterzugeben. Vollständige Cookie-Anleitung.
