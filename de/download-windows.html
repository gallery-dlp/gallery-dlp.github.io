Schnellstart

Nach dem Herunterladen von gallery-dl.exe

Vier Schritte vom Download bis zur ersten Bildergalerie.

Schritt 01

In Ordner speichern

gallery-dl.exe in dauerhaften Ordner ablegen, z. B. C:\Tools\gallery-dl\ — Downloads meiden, der kann automatisch geleert werden.

Schritt 02

Browser-Warnung

In Edge/Chrome auf Behalten → Trotzdem behalten klicken. Bei SmartScreen: Weitere Informationen → Trotzdem ausführen.

Schritt 03

PowerShell im Ordner öffnen

Explorer-Adressleiste anklicken, powershell eingeben, Enter drücken. PowerShell öffnet sich direkt im richtigen Ordner.

Schritt 04

Überprüfen & ausführen

gallery-dl --version eingeben zur Bestätigung, dann beliebige URL einfügen und loslegen.

FAQ

Häufige Fragen zum Download

Benötige ich Administratorrechte für gallery-dl.exe?
Nein. gallery-dl.exe ist portabel — keine Installation erforderlich. Administratorrechte werden nur benötigt, um den Ordner zum System-PATH für alle Benutzer hinzuzufügen.
Benötige ich Python für gallery-dl unter Windows?
Nein. Die gallery-dl.exe-Version enthält intern eine Python-Laufzeitumgebung. Einfach die .exe herunterladen und ausführen — keine Python-Installation nötig.
Wie lade ich von einem privaten Instagram- oder Twitter-Konto herunter?
Cookies vom Browser weitergeben: gallery-dl --cookies-from-browser chrome "URL". Vollständige Cookie-Anleitung.
Wie halte ich gallery-dl aktuell?
PowerShell im gallery-dl.exe-Ordner öffnen und gallery-dl -U ausführen. Automatisches Update auf die neueste Version.
