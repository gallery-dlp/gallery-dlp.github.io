Instalación paso a paso

Instalar gallery-dl en Windows en 5 pasos

Funciona en Windows 11 y Windows 10 (64 bits). Sin asistente de instalación, sin derechos de administrador.

Paso 01

Descargar gallery-dl.exe

Ve a la página de descarga y guarda gallery-dl.exe desde la versión oficial de GitHub.

Paso 02

Guardar en carpeta dedicada

Mueve el archivo a una ubicación permanente, ej: C:\Tools\gallery-dl\. Evita Descargas — los archivos allí pueden borrarse automáticamente.

Paso 03

Abrir PowerShell en esa carpeta

En el Explorador, haz clic en la barra de direcciones, escribe powershell y pulsa Intro. PowerShell se abre directamente en la carpeta correcta.

Paso 04

Verificar que funciona

Ejecuta gallery-dl --version. Debería aparecer un número de versión como 1.28.x. Si es así, funciona.

Paso 05

Actualizar a la última versión

Ejecuta gallery-dl -U. Descarga y reemplaza automáticamente por la última versión. Hazlo regularmente.

Opcional

Configurar archivo de configuración

Crea un gallery-dl.conf para definir carpetas de salida por defecto, plantillas de nombre de archivo y cookies. Guía de configuración →

Opcional pero recomendado

Cómo añadir gallery-dl al PATH en Windows

Con gallery-dl en el PATH puedes ejecutar gallery-dl desde cualquier carpeta en PowerShell o CMD.

Método gráfico (recomendado)

  1. Pulsa Win y busca Variables de entorno
  2. Haz clic en Editar las variables de entorno del sistema
  3. En Variables de usuario, selecciona Path → Editar
  4. Haz clic en Nuevo y pega la ruta, ej: C:\Tools\gallery-dl
  5. Haz clic en Aceptar en todos los diálogos y reinicia el terminal

Método PowerShell

Ejecuta una vez en PowerShell — define el PATH solo para tu usuario, sin derechos de admin:

setx PATH "%PATH%;C:\Tools\gallery-dl"

Cierra y vuelve a abrir PowerShell. Prueba con gallery-dl --version desde cualquier directorio.

Nota: Si ves «gallery-dl no se reconoce» después de añadir al PATH, cierra todas las ventanas del terminal y vuelve a abrirlas — los cambios del PATH solo tienen efecto en nuevas sesiones.

Problemas comunes

Solucionar problemas de instalación de gallery-dl en Windows

Advertencia de Windows SmartScreen al ejecutar gallery-dl.exe
Haz clic en Más información → Ejecutar de todos modos. SmartScreen señala los ejecutables nuevos por defecto. gallery-dl.exe es la versión oficial de código abierto sin modificar de GitHub. Alternativamente: clic derecho → Propiedades → marca Desbloquear en la parte inferior → Aceptar.
«gallery-dl no se reconoce» en PowerShell
Dos opciones: ejecuta gallery-dl directamente desde su carpeta, o añade la carpeta al PATH (ver sección anterior). Después de añadir al PATH, cierra todas las ventanas del terminal y vuelve a abrirlas.
«Acceso denegado» o archivo bloqueado por el antivirus
Clic derecho en gallery-dl.exe → Propiedades → marca Desbloquear en la parte inferior si es visible → Aceptar. Si el antivirus bloquea, añade una excepción — es un falso positivo para portables de código abierto.
gallery-dl no descarga nada / se cierra inmediatamente
El sitio probablemente requiere inicio de sesión. Ejecuta gallery-dl --cookies-from-browser chrome "URL". Ver la guía de cookies para detalles completos.
