gallery-dl DeviantArt en Windows — Descargar Galerías & Favoritos

gallery-dl descarga galerías completas de usuarios, favoritos, colecciones y deviations individuales de DeviantArt en resolución original. Las galerías públicas funcionan sin inicio de sesión — el contenido maduro requiere cookies.

⬇ Descargar gallery-dl.exe Guía de configuración de cookies →

Comandos

Comandos gallery-dl DeviantArt para Windows

Ejecuta en PowerShell desde la carpeta que contiene gallery-dl.exe. Las galerías públicas funcionan sin cookies.

Descargar galería completa (pública — sin inicio de sesión)
gallery-dl "https://www.deviantart.com/username/gallery/"
Descargar galería con inicio de sesión (para contenido maduro)
gallery-dl --cookies-from-browser chrome "https://www.deviantart.com/username/gallery/"
Descargar favoritos
gallery-dl --cookies-from-browser chrome "https://www.deviantart.com/username/favourites/"
Descargar una carpeta de colección específica
gallery-dl --cookies-from-browser chrome "https://www.deviantart.com/username/gallery/FOLDER_ID/nombre-carpeta"
Descargar una deviation individual
gallery-dl "https://www.deviantart.com/username/art/nombre-obra-123456789"
Guardar en carpeta personalizada
gallery-dl -d "C:\Imagenes\DeviantArt" "https://www.deviantart.com/username/gallery/"

Configuración para DeviantArt

Configuración DeviantArt recomendada para gallery-dl en Windows

config.json — sección DeviantArt
{ "extractor": { "deviantart": { "cookies": "C:/Tools/gallery-dl/deviantart-cookies.txt", "original": true, "metadata": true, "filename": "{author}_{index}_{title}.{extension}", "sleep-request": 1.0 } } }
original: true
Descargar el archivo original enviado (sin redimensionar)
metadata: true
Guardar metadatos de la deviation como JSON

Preguntas frecuentes

gallery-dl DeviantArt — preguntas frecuentes

¿gallery-dl descarga DeviantArt en resolución original?
Sí, cuando "original": true está definido en la configuración (o con sesión iniciada). Sin esto, gallery-dl puede descargar la versión de visualización web. Algunas deviations antiguas solo tienen versión web disponible.
gallery-dl DeviantArt dice «filtro de contenido» u omite deviations
El contenido maduro se filtra para los visitantes no autenticados. Inicia sesión en DeviantArt en el navegador y pasa las cookies: --cookies-from-browser chrome. Asegúrate de que el contenido maduro esté habilitado en la configuración de tu cuenta de DeviantArt.
gallery-dl está descargando DeviantArt muy lentamente
DeviantArt tiene límites de velocidad. El valor por defecto sleep-request: 1.0 generalmente es suficiente. Si obtienes errores 429, auméntalo a 2.0 o 3.0. No ejecutes múltiples instancias simultáneas de gallery-dl en DeviantArt.
¿Cómo descargo solo una carpeta específica de una galería de DeviantArt?
Navega hasta la carpeta en DeviantArt, copia la URL (contiene el ID de carpeta) y pásala directamente a gallery-dl. El formato de URL es deviantart.com/username/gallery/FOLDER_ID/nombre-carpeta.
