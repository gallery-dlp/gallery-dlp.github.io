Referencia de Comandos gallery-dl para Windows

Todos los comandos esenciales de gallery-dl en un solo lugar. Pega directamente en PowerShell o CMD. Sustituye la URL por tu enlace de destino.

Esenciales

Comandos básicos de gallery-dl

Descargar todas las imágenes de una URL
gallery-dl "URL"
Descargar con cookies del navegador (para sitios que requieren inicio de sesión)
gallery-dl --cookies-from-browser chrome "URL"
Verificar la versión instalada
gallery-dl --version
Actualizar a la última versión
gallery-dl -U
Simular — listar imágenes que se descargarían (sin descargar)
gallery-dl --simulate "URL"
Salida detallada para depuración
gallery-dl -v "URL"

Salida & nombres de archivo

Definir carpeta de descarga y plantillas de nombre de archivo

Guardar en carpeta específica
gallery-dl -d "C:\Imagenes" "URL"
Nombre de archivo personalizado — categoría + autor + número secuencial
gallery-dl --filename "{category}_{author}_{num:>04}.{extension}" "URL"
Crear subcarpetas por sitio y autor
gallery-dl -d "C:\Imagenes" --directory "{category}/{author}" "URL"
Usar archivo de configuración personalizado
gallery-dl --config "C:\Tools\gallery-dl\miconfig.json" "URL"

Filtros & selección

Filtrar descargas por tipo, fecha o cantidad

Descargar solo imágenes (ignorar vídeos)
gallery-dl --image-filter "extension in ('jpg','jpeg','png','webp','gif')" "URL"
Descargar solo las primeras 50 imágenes
gallery-dl --chapter-filter "index < 50" "URL"
Omitir ya descargados — usar archivo de historial
gallery-dl --download-archive "C:\Tools\gallery-dl\archivo.txt" "URL"
Descargar solo imágenes de más de 500 KB
gallery-dl --image-filter "filesize > 500000" "URL"

Rendimiento & rate limiting

Velocidad, reintentos y rate limiting

Limitar velocidad de descarga
gallery-dl --rate 2M "URL"
Pausa entre peticiones (segundos) — evita rate limits
gallery-dl --sleep-request 2 "URL"
Definir número de reintentos en caso de fallo
gallery-dl --retries 5 "URL"
Descarga masiva desde archivo de texto (una URL por línea)
gallery-dl --input-file "C:\Tools\gallery-dl\enlaces.txt"

Variables de nombre de archivo

Variables de plantilla comunes para --filename y --directory

{filename}
Nombre de archivo original
{extension}
Extensión de archivo
{category}
Nombre del sitio (instagram, pixiv…)
{author}
Nombre del autor / uploader
{num}
Número secuencial
{num:>04}
Número con ceros (0001…)
{date}
Fecha de publicación/subida
{date:%Y-%m-%d}
Fecha formateada (2026-06-09)
{id}
ID de la publicación / imagen
