Windows 11/10  ·  Erreurs courantes  ·  Solutions étape par étape

gallery-dl Ne Fonctionne Pas sur Windows — Corriger les Erreurs Courantes

La plupart des erreurs gallery-dl sur Windows ont des solutions simples. Commencez ici : mettez à jour avec gallery-dl -U et vérifiez les cookies. Cela résout 80% des problèmes.

Commencez toujours ici :

Étape 1 — mettre à jour gallery-dl
gallery-dl -U
Étape 2 — sortie détaillée pour voir l'erreur complète
gallery-dl -v "URL"

Référence des erreurs

Messages d'erreur gallery-dl et comment les corriger

« Login required » ou « Authentication required »
Cause : Le site nécessite que vous soyez connecté pour accéder à ce contenu.

Solution : Connectez-vous au site dans Chrome (ou Firefox/Edge), fermez complètement le navigateur, puis exécutez : gallery-dl --cookies-from-browser chrome "URL" Si les cookies continuent d'expirer, exportez un fichier cookies.txt. Voir le guide des cookies.
« gallery-dl n'est pas reconnu comme commande »
Cause : PowerShell ne trouve pas gallery-dl.exe car son dossier n'est pas dans le PATH.

Solution — option 1 : Naviguez jusqu'au dossier contenant gallery-dl.exe dans l'Explorateur, cliquez sur la barre d'adresse, tapez powershell, appuyez sur Entrée. Exécutez gallery-dl de là.

Solution — option 2 : Ajoutez le dossier gallery-dl au PATH. Voir le guide d'installation.
Erreur HTTP 403 Forbidden
Cause : Le site bloque la requête — rate limiting, connexion requise ou blocage géographique.

Solution : Mettez d'abord à jour : gallery-dl -U. Puis essayez avec les cookies : gallery-dl --cookies-from-browser chrome "URL" Si vous utilisez un VPN, essayez sans. Ajoutez --sleep-request 3 si vous atteignez les limites de débit.
gallery-dl ne télécharge rien — se ferme immédiatement
Cause : Connexion requise, format d'URL non supporté ou extracteur nécessite une mise à jour.

Solution : Exécutez avec sortie détaillée : gallery-dl -v "URL" Puis essayez avec les cookies : gallery-dl --cookies-from-browser chrome "URL". Mettez à jour : gallery-dl -U.
Erreur de certificat SSL sur Windows
Cause : Le magasin de certificats Windows est obsolète, ou un proxy/VPN intercepte HTTPS.

Solution : Exécutez Windows Update (Paramètres → Windows Update), puis réessayez. En dernier recours seulement : gallery-dl --no-check-certificate "URL" — déconseillé à long terme.
Windows SmartScreen bloque gallery-dl.exe
Cause : SmartScreen signale les exécutables portables non signés par défaut — c'est un faux positif.

Solution : Cliquez sur Informations complémentaires → Exécuter quand même. Ou clic droit sur gallery-dl.exe → Propriétés → cochez Débloquer en bas → OK.
« database is locked » lors de l'utilisation de --cookies-from-browser
Cause : Chrome ou Edge verrouille sa base de données SQLite de cookies pendant son exécution.

Solution : Fermez complètement le navigateur — vérifiez aussi l'icône dans la barre des tâches. Puis réexécutez gallery-dl. Alternativement, exportez les cookies vers un fichier cookies.txt et utilisez --cookies.
gallery-dl est très lent — les téléchargements se bloquent ou s'arrêtent
Cause : Rate limiting du site, connexion instable ou VPN throttling.

Solution : Ajoutez --sleep-request 3 --retries 5 à votre commande. Pour les blocages persistants, essayez --part pour reprendre les téléchargements partiels.
