gallery-dl Instagram sur Windows — Télécharger Photos, Reels & Stories

gallery-dl télécharge des profils complets, publications, Reels, Stories et highlights Instagram en résolution originale. La connexion via les cookies du navigateur est obligatoire depuis les restrictions d'accès public d'Instagram en 2023.

⬇ Télécharger gallery-dl.exe Guide de configuration des cookies →
Mise à jour 2026 — connexion obligatoire : Depuis mi-2023, Instagram nécessite une connexion pour presque tout le contenu. Passez vos cookies du navigateur avec --cookies-from-browser chrome. Assurez-vous d'être connecté à Instagram dans votre navigateur d'abord.

Commandes

Commandes gallery-dl Instagram pour Windows

Toutes les commandes exécutées dans PowerShell depuis le dossier contenant gallery-dl.exe.

Télécharger toutes les publications d'un profil public ou suivi
gallery-dl --cookies-from-browser chrome "https://www.instagram.com/username/"
Télécharger une publication unique
gallery-dl --cookies-from-browser chrome "https://www.instagram.com/p/POST_ID/"
Télécharger les Stories (connexion obligatoire)
gallery-dl --cookies-from-browser chrome "https://www.instagram.com/stories/username/"
Télécharger les Highlights
gallery-dl --cookies-from-browser chrome "https://www.instagram.com/highlights/HIGHLIGHT_ID/"
Sauvegarder dans un dossier spécifique avec sous-dossier par nom d'utilisateur
gallery-dl --cookies-from-browser chrome -d "C:\Images\Instagram" "https://www.instagram.com/username/"
Télécharger avec fichier cookies.txt sauvegardé (pour l'automatisation)
gallery-dl --cookies "C:\Tools\gallery-dl\instagram-cookies.txt" "https://www.instagram.com/username/"

Configuration pour Instagram

Définir les cookies Instagram de façon permanente dans la configuration gallery-dl

config.json — section Instagram
{ "extractor": { "base-directory": "C:/Images/", "instagram": { "cookies": "C:/Tools/gallery-dl/instagram-cookies.txt", "sleep-request": 3.0, "videos": true, "filename": "{username}_{shortcode}_{num}.{extension}" } } }

sleep-request: 3.0 — ralentit les requêtes pour éviter le rate limiting d'Instagram. Recommandé pour les téléchargements de grands profils.

FAQ

gallery-dl Instagram — questions fréquentes

gallery-dl fonctionne-t-il encore avec Instagram en 2026 ?
Oui, mais la connexion est obligatoire. Instagram a restreint l'accès public en 2023. L'extracteur Instagram de gallery-dl est activement maintenu. Maintenez à jour avec gallery-dl -U.
gallery-dl dit « Connexion Instagram requise » même avec des cookies
Vos cookies ont peut-être expiré. Déconnectez-vous et reconnectez-vous à Instagram dans le navigateur, puis réexécutez avec --cookies-from-browser chrome. Assurez-vous que le navigateur est complètement fermé avant d'exécuter gallery-dl sur Windows.
gallery-dl Instagram est très lent — comment accélérer ?
Instagram limite les scrapers agressifs. Ne réduisez pas sleep-request en dessous de 2 secondes ou votre compte pourrait être temporairement restreint. Utilisez --download-archive archive.txt pour ignorer les fichiers déjà téléchargés.
gallery-dl peut-il télécharger des Reels Instagram ?
Oui. Les Reels sont téléchargés en fichiers vidéo .mp4. Utilisez l'URL du profil pour tous les Reels d'un utilisateur, ou l'URL directe du Reel pour une seule vidéo. La connexion via cookies est obligatoire.
Comment ignorer les fichiers déjà téléchargés ?
Utilisez le flag d'archive de téléchargement : gallery-dl --download-archive archive.txt --cookies-from-browser chrome "URL". gallery-dl enregistre chaque ID de fichier téléchargé dans l'archive et l'ignore lors des exécutions futures.
