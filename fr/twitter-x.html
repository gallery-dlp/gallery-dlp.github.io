gallery-dl Twitter/X sur Windows — Télécharger Photos & Vidéos

gallery-dl télécharge des images et vidéos en haute résolution depuis les timelines, onglets médias, likes et tweets individuels de Twitter/X. L'authentification par cookie est nécessaire depuis les restrictions API de Twitter.

⬇ Télécharger gallery-dl.exe Guide de configuration des cookies →
Important — les cookies expirent rapidement : Les cookies Twitter/X expirent plus fréquemment que les autres sites. Si gallery-dl arrête soudainement de télécharger, réexécutez simplement avec --cookies-from-browser chrome pour actualiser la session.

Commandes

Commandes gallery-dl Twitter/X pour Windows

Télécharger tous les médias depuis l'onglet médias d'un utilisateur
gallery-dl --cookies-from-browser chrome "https://x.com/username/media"
Télécharger les médias d'un tweet unique
gallery-dl --cookies-from-browser chrome "https://x.com/username/status/TWEET_ID"
Télécharger la timeline d'un utilisateur
gallery-dl --cookies-from-browser chrome "https://x.com/username"
Télécharger l'onglet likes
gallery-dl --cookies-from-browser chrome "https://x.com/username/likes"
Télécharger avec dossier de sortie personnalisé
gallery-dl --cookies-from-browser chrome -d "C:\Images\Twitter" "https://x.com/username/media"
Télécharger avec cookies.txt (pour les tâches planifiées)
gallery-dl --cookies "C:\Tools\gallery-dl\twitter-cookies.txt" "https://x.com/username/media"

Configuration pour Twitter/X

Configuration Twitter/X recommandée pour gallery-dl sur Windows

config.json — section Twitter
{ "extractor": { "twitter": { "cookies": "C:/Tools/gallery-dl/twitter-cookies.txt", "videos": true, "retweets": false, "quoted": true, "filename": "{author[name]}_{tweet_id}_{num}.{extension}", "sleep-request": 1.0 } } }
videos: true
Télécharger aussi les fichiers vidéo
retweets: false
Ignorer les retweets — publications originales seulement
quoted: true
Inclure les médias des tweets cités
sleep-request: 1.0
Pause entre les requêtes pour éviter le rate limiting

FAQ

gallery-dl Twitter/X — questions fréquentes

gallery-dl fonctionne-t-il avec Twitter/X en 2026 ?
Oui. L'extracteur Twitter/X de gallery-dl est activement maintenu. Depuis les restrictions API de Twitter en 2023, la connexion via cookies est obligatoire. Maintenez à jour avec gallery-dl -U.
gallery-dl Twitter dit « authentification requise » avec des cookies
Les cookies Twitter/X expirent fréquemment. Déconnectez-vous et reconnectez-vous sur x.com dans le navigateur, fermez complètement le navigateur, puis réexécutez avec --cookies-from-browser chrome.
gallery-dl télécharge des images Twitter en basse résolution
Par défaut, gallery-dl ajoute ?format=jpg&name=orig pour demander la résolution originale. Si vous obtenez toujours des images en basse résolution, mettez à jour avec gallery-dl -U.
Comment télécharger uniquement les images et ignorer les vidéos ?
Ajoutez un filtre d'image : gallery-dl --image-filter "extension in ('jpg','png','webp','gif')" --cookies-from-browser chrome "URL". Ou définissez "videos": false dans la section Twitter de config.json.
gallery-dl peut-il télécharger depuis des comptes Twitter privés/protégés ?
Oui, si vous suivez le compte et êtes connecté. Passez vos cookies du navigateur — gallery-dl accédera à la timeline en utilisant votre session authentifiée.
