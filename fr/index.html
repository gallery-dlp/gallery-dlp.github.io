Commandes gallery-dl pour Windows

Commandes gallery-dl les plus utiles — copiez & collez

Exécutez directement dans PowerShell ou CMD. Remplacez l'URL par votre lien cible.

Télécharger toutes les images depuis une URL
gallery-dl "https://www.instagram.com/username/"
Télécharger avec les cookies du navigateur (pour les sites nécessitant une connexion)
gallery-dl --cookies-from-browser chrome "URL"
Sauvegarder dans un dossier personnalisé avec les noms de fichiers originaux
gallery-dl -d "C:\Images\%(category)s\%(uploader)s" "URL"
Télécharger uniquement les images (ignorer les vidéos)
gallery-dl --image-filter "extension in ('jpg','png','webp')" "URL"

FAQ

Questions fréquentes sur gallery-dl pour Windows

gallery-dl est-il sûr sur Windows ?
Oui. gallery-dl est entièrement open source et hébergé sur GitHub. Chaque ligne de code est auditée publiquement. Aucun logiciel supplémentaire, aucune télémétrie.
gallery-dl fonctionne-t-il sur Windows 11 ?
Oui. gallery-dl.exe fonctionne sur Windows 11 et Windows 10 (64 bits) sans installation. Téléchargez le .exe, ouvrez PowerShell et exécutez votre première commande.
Quels sites gallery-dl supporte-t-il ?
gallery-dl supporte 200+ sites dont Instagram, Twitter/X, Pixiv, DeviantArt, Reddit, Tumblr, Danbooru, Flickr et bien d'autres. Voir la liste complète sur GitHub.
Ai-je besoin de Python pour utiliser gallery-dl sur Windows ?
Non. La version gallery-dl.exe inclut un runtime Python intégré. Téléchargez simplement le .exe et exécutez — aucune installation de Python nécessaire.
Comment mettre à jour gallery-dl sur Windows ?
Ouvrez PowerShell dans le dossier gallery-dl et exécutez gallery-dl -U. Mise à jour automatique vers la dernière version.
Pourquoi gallery-dl dit « connexion requise » ou « 403 Forbidden » ?
De nombreux sites nécessitent une authentification pour le contenu privé. Utilisez --cookies-from-browser chrome pour passer votre session navigateur. Voir le guide complet des cookies.
