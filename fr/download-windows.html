Démarrage rapide

Après avoir téléchargé gallery-dl.exe

Quatre étapes du téléchargement jusqu'à votre première galerie d'images.

Étape 01

Sauvegarder dans un dossier

Placez gallery-dl.exe dans un dossier permanent comme C:\Tools\gallery-dl\ — évitez Téléchargements qui peut être vidé automatiquement.

Étape 02

Si le navigateur avertit

Cliquez sur Conserver → Conserver quand même. Si SmartScreen apparaît : Informations complémentaires → Exécuter quand même.

Étape 03

Ouvrir PowerShell dans le dossier

Cliquez sur la barre d'adresse de l'Explorateur, tapez powershell et appuyez sur Entrée. PowerShell s'ouvre directement dans le bon dossier.

Étape 04

Vérifier & exécuter

Tapez gallery-dl --version pour confirmer, puis collez n'importe quelle URL pour commencer à télécharger.

FAQ

FAQ sur le téléchargement

Ai-je besoin de droits administrateur pour exécuter gallery-dl.exe ?
Non. gallery-dl.exe est portable — aucune installation nécessaire. Les droits administrateur ne sont nécessaires que pour ajouter le dossier au PATH système pour tous les utilisateurs.
Ai-je besoin de Python installé pour utiliser gallery-dl sur Windows ?
Non. La version gallery-dl.exe inclut un runtime Python en interne. Téléchargez simplement le .exe et exécutez — aucune installation de Python nécessaire.
Comment télécharger depuis un compte Instagram ou Twitter privé ?
Passez les cookies du navigateur : gallery-dl --cookies-from-browser chrome "URL". Voir le guide complet des cookies et de la connexion.
Comment maintenir gallery-dl à jour ?
Ouvrez PowerShell dans le dossier gallery-dl.exe et exécutez gallery-dl -U. Mise à jour automatique vers la dernière version.
