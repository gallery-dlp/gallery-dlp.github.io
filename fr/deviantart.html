gallery-dl DeviantArt sur Windows — Télécharger Galeries & Favoris

gallery-dl télécharge des galeries complètes d'utilisateurs, favoris, collections et deviations individuelles de DeviantArt en résolution originale. Les galeries publiques fonctionnent sans connexion — le contenu mature nécessite des cookies.

⬇ Télécharger gallery-dl.exe Guide de configuration des cookies →

Commandes

Commandes gallery-dl DeviantArt pour Windows

Exécutez dans PowerShell depuis le dossier contenant gallery-dl.exe. Les galeries publiques fonctionnent sans cookies.

Télécharger la galerie complète (publique — sans connexion)
gallery-dl "https://www.deviantart.com/username/gallery/"
Télécharger la galerie avec connexion (pour le contenu mature)
gallery-dl --cookies-from-browser chrome "https://www.deviantart.com/username/gallery/"
Télécharger les favoris
gallery-dl --cookies-from-browser chrome "https://www.deviantart.com/username/favourites/"
Télécharger un dossier de collection spécifique
gallery-dl --cookies-from-browser chrome "https://www.deviantart.com/username/gallery/FOLDER_ID/nom-dossier"
Télécharger une deviation individuelle
gallery-dl "https://www.deviantart.com/username/art/nom-oeuvre-123456789"
Sauvegarder dans un dossier personnalisé
gallery-dl -d "C:\Images\DeviantArt" "https://www.deviantart.com/username/gallery/"

Configuration pour DeviantArt

Configuration DeviantArt recommandée pour gallery-dl sur Windows

config.json — section DeviantArt
{ "extractor": { "deviantart": { "cookies": "C:/Tools/gallery-dl/deviantart-cookies.txt", "original": true, "metadata": true, "filename": "{author}_{index}_{title}.{extension}", "sleep-request": 1.0 } } }
original: true
Télécharger le fichier original envoyé (non redimensionné)
metadata: true
Sauvegarder les métadonnées de la deviation en JSON

FAQ

gallery-dl DeviantArt — questions fréquentes

gallery-dl télécharge-t-il DeviantArt en résolution originale ?
Oui, quand "original": true est défini dans la configuration (ou avec une session connectée). Sans cela, gallery-dl peut télécharger la version d'affichage web. Certaines anciennes deviations n'ont que la version web disponible.
gallery-dl DeviantArt dit « filtre de contenu » ou ignore des deviations
Le contenu mature est filtré pour les visiteurs non connectés. Connectez-vous à DeviantArt dans le navigateur et passez les cookies : --cookies-from-browser chrome. Assurez-vous que le contenu mature est activé dans les paramètres du compte DeviantArt.
gallery-dl télécharge DeviantArt très lentement
DeviantArt a des limites de débit. Le paramètre par défaut sleep-request: 1.0 est généralement suffisant. Si vous obtenez des erreurs 429, augmentez à 2.0 ou 3.0. N'exécutez pas plusieurs instances simultanées de gallery-dl sur DeviantArt.
Comment télécharger uniquement un dossier spécifique d'une galerie DeviantArt ?
Naviguez jusqu'au dossier sur DeviantArt, copiez l'URL (elle contient l'ID du dossier) et passez-la directement à gallery-dl. Le format de l'URL est deviantart.com/username/gallery/FOLDER_ID/nom-dossier.
