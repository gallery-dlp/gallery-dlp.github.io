Référence des Commandes gallery-dl pour Windows

Toutes les commandes essentielles de gallery-dl en un seul endroit. Collez directement dans PowerShell ou CMD. Remplacez l'URL par votre lien cible.

Essentiels

Commandes de base gallery-dl

Télécharger toutes les images depuis une URL
gallery-dl "URL"
Télécharger avec les cookies du navigateur (pour les sites nécessitant une connexion)
gallery-dl --cookies-from-browser chrome "URL"
Vérifier la version installée
gallery-dl --version
Mettre à jour vers la dernière version
gallery-dl -U
Simuler — lister les images qui seraient téléchargées (sans télécharger)
gallery-dl --simulate "URL"
Sortie détaillée pour le débogage
gallery-dl -v "URL"

Sortie & noms de fichiers

Définir le dossier de téléchargement et les modèles de noms de fichiers

Sauvegarder dans un dossier spécifique
gallery-dl -d "C:\Images" "URL"
Nom de fichier personnalisé — catégorie + auteur + numéro séquentiel
gallery-dl --filename "{category}_{author}_{num:>04}.{extension}" "URL"
Créer des sous-dossiers par site et auteur
gallery-dl -d "C:\Images" --directory "{category}/{author}" "URL"
Utiliser un fichier de configuration personnalisé
gallery-dl --config "C:\Tools\gallery-dl\maconfig.json" "URL"

Filtres & sélection

Filtrer les téléchargements par type, date ou quantité

Télécharger uniquement les images (ignorer les vidéos)
gallery-dl --image-filter "extension in ('jpg','jpeg','png','webp','gif')" "URL"
Télécharger uniquement les 50 premières images
gallery-dl --chapter-filter "index < 50" "URL"
Ignorer les déjà téléchargés — utiliser un fichier d'archive
gallery-dl --download-archive "C:\Tools\gallery-dl\archive.txt" "URL"
Télécharger uniquement les images de plus de 500 Ko
gallery-dl --image-filter "filesize > 500000" "URL"

Performance & rate limiting

Vitesse, nouvelles tentatives et rate limiting

Limiter la vitesse de téléchargement
gallery-dl --rate 2M "URL"
Pause entre les requêtes (secondes) — évite le rate limiting
gallery-dl --sleep-request 2 "URL"
Définir le nombre de tentatives en cas d'échec
gallery-dl --retries 5 "URL"
Téléchargement en masse depuis un fichier texte (une URL par ligne)
gallery-dl --input-file "C:\Tools\gallery-dl\liens.txt"

Variables de noms de fichiers

Variables de modèles courantes pour --filename et --directory

{filename}
Nom de fichier original
{extension}
Extension de fichier
{category}
Nom du site (instagram, pixiv…)
{author}
Nom de l'auteur / uploader
{num}
Numéro séquentiel
{num:>04}
Numéro avec zéros (0001…)
{date}
Date de publication/upload
{date:%Y-%m-%d}
Date formatée (2026-06-09)
{id}
ID de la publication / image
