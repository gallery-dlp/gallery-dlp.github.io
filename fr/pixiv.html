gallery-dl Pixiv sur Windows — Télécharger Œuvres, Galeries & Favoris

gallery-dl télécharge des galeries d'artistes Pixiv complètes, vos favoris, recherches par tag et animations ugoira. Un compte Pixiv gratuit et une connexion par cookie sont nécessaires pour un accès complet.

Commandes

Commandes gallery-dl Pixiv pour Windows

Télécharger la galerie complète d'un artiste
gallery-dl --cookies-from-browser chrome "https://www.pixiv.net/users/USER_ID"
Télécharger une œuvre individuelle (toutes les pages)
gallery-dl --cookies-from-browser chrome "https://www.pixiv.net/artworks/ARTWORK_ID"
Télécharger vos favoris
gallery-dl --cookies-from-browser chrome "https://www.pixiv.net/users/VOTRE_USER_ID/bookmarks/artworks"
Télécharger des œuvres par tag
gallery-dl --cookies-from-browser chrome "https://www.pixiv.net/tags/NOM_TAG/artworks"
Télécharger le fil des abonnements (nouvelles œuvres des artistes suivis)
gallery-dl --cookies-from-browser chrome "https://www.pixiv.net/bookmark_new_illust.php"
Sauvegarder dans un dossier personnalisé avec sous-dossier par nom d'artiste
gallery-dl --cookies-from-browser chrome -d "C:\Images\Pixiv" "https://www.pixiv.net/users/USER_ID"

Animations Ugoira

Comment télécharger des ugoira Pixiv avec gallery-dl sur Windows

Les ugoira sont les animations basées sur des frames de Pixiv. gallery-dl peut télécharger le ZIP brut ou convertir en GIF/WebM avec FFmpeg.

Par défaut — télécharger en ZIP

gallery-dl télécharge les ugoira en archive ZIP contenant des frames individuels. Aucun FFmpeg nécessaire.

gallery-dl --cookies-from-browser chrome "https://www.pixiv.net/artworks/UGOIRA_ID"

Convertir en GIF ou WebM (FFmpeg requis)

Ajoutez ceci à config.json pour convertir automatiquement les ugoira :

"pixiv": { "ugoira": true, "ffmpeg-args": ["-vf", "scale=600:-1"], "ffmpeg-output": "webm" }

FAQ

gallery-dl Pixiv — questions fréquentes

gallery-dl nécessite-t-il un compte Pixiv ?
Oui. Un compte Pixiv gratuit est nécessaire. Sans connexion, gallery-dl ne peut accéder qu'à un petit sous-ensemble d'œuvres publiques. Connectez-vous sur pixiv.net dans votre navigateur, puis utilisez --cookies-from-browser chrome.
gallery-dl ignore les œuvres R-18 de Pixiv — comment résoudre ?
Le contenu R-18 nécessite un compte Pixiv avec le contenu adulte activé dans les paramètres du compte. Connectez-vous sur pixiv.net, activez R-18 dans les paramètres, puis passez les cookies à gallery-dl.
Comment trouver l'ID utilisateur Pixiv d'un artiste ?
Ouvrez le profil de l'artiste sur Pixiv dans votre navigateur. L'URL ressemble à pixiv.net/users/12345678 — le nombre est l'ID utilisateur. Vous pouvez utiliser l'URL complète directement avec gallery-dl.
gallery-dl télécharge les ugoira en ZIP au lieu de GIF — comment convertir ?
Placez FFmpeg dans le dossier gallery-dl (C:\Tools\gallery-dl\ffmpeg.exe) et définissez "ugoira": true et "ffmpeg-output": "gif" dans la section Pixiv du config.json. gallery-dl convertira alors automatiquement les ugoira en GIF ou WebM.
Comment télécharger uniquement les illustrations (sans manga/ugoira) ?
Utilisez un filtre d'image : gallery-dl --image-filter "type == 'illust'" --cookies-from-browser chrome "URL". Les types sont : illust, manga, ugoira.
