Referência de Comandos gallery-dl para Windows

Todos os comandos essenciais do gallery-dl em um só lugar. Cole diretamente no PowerShell ou CMD. Substitua a URL pelo seu link de destino.

Essenciais

Comandos básicos do gallery-dl

Baixar todas as imagens de uma URL
gallery-dl "URL"
Baixar com cookies do navegador (para sites que requerem login)
gallery-dl --cookies-from-browser chrome "URL"
Verificar versão instalada
gallery-dl --version
Atualizar para a versão mais recente
gallery-dl -U
Simular — listar imagens que seriam baixadas (sem baixar)
gallery-dl --simulate "URL"
Saída detalhada para depuração
gallery-dl -v "URL"

Saída & nomes de arquivo

Definir pasta de download e modelos de nome de arquivo

Salvar em pasta específica
gallery-dl -d "C:\Imagens" "URL"
Nome de arquivo personalizado — categoria + autor + número sequencial
gallery-dl --filename "{category}_{author}_{num:>04}.{extension}" "URL"
Criar subpastas por site e autor
gallery-dl -d "C:\Imagens" --directory "{category}/{author}" "URL"
Usar arquivo de configuração personalizado
gallery-dl --config "C:\Tools\gallery-dl\meuconfig.json" "URL"

Autenticação

Comandos de cookies e login

Usar cookies do Chrome
gallery-dl --cookies-from-browser chrome "URL"
Usar cookies do Firefox
gallery-dl --cookies-from-browser firefox "URL"
Usar arquivo cookies.txt
gallery-dl --cookies "C:\Tools\gallery-dl\cookies.txt" "URL"
Login com usuário e senha (onde suportado)
gallery-dl -u "usuario" -p "senha" "URL"

Filtros & seleção

Filtrar downloads por tipo, data ou quantidade

Baixar apenas imagens (pular vídeos)
gallery-dl --image-filter "extension in ('jpg','jpeg','png','webp','gif')" "URL"
Baixar apenas as primeiras 50 imagens
gallery-dl --chapter-filter "index < 50" "URL"
Pular já baixados — usar arquivo de arquivo
gallery-dl --download-archive "C:\Tools\gallery-dl\arquivo.txt" "URL"
Baixar apenas imagens maiores que 500 KB
gallery-dl --image-filter "filesize > 500000" "URL"

Desempenho & rate limiting

Velocidade, novas tentativas e rate limiting

Limitar velocidade de download
gallery-dl --rate 2M "URL"
Pausa entre requisições (segundos) — evita rate limits
gallery-dl --sleep-request 2 "URL"
Definir número de tentativas em caso de falha
gallery-dl --retries 5 "URL"
Download em massa de arquivo de texto (uma URL por linha)
gallery-dl --input-file "C:\Tools\gallery-dl\links.txt"

Variáveis de nome de arquivo

Variáveis de modelo comuns para --filename e --directory

{filename}
Nome de arquivo original
{extension}
Extensão do arquivo
{category}
Nome do site (instagram, pixiv…)
{author}
Nome do autor / uploader
{num}
Número sequencial
{num:>04}
Número com zeros (0001…)
{date}
Data da postagem/upload
{date:%Y-%m-%d}
Data formatada (2026-06-09)
{id}
ID da postagem / imagem
