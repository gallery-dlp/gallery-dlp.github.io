gallery-dl Twitter/X no Windows — Baixar Fotos & Vídeos

gallery-dl baixa imagens e vídeos em resolução completa de linhas do tempo, abas de mídia, curtidas e tweets individuais do Twitter/X. Autenticação por cookie é necessária desde as restrições de API do Twitter.

⬇ Baixar gallery-dl.exe Guia de configuração de cookies →
Importante — cookies expiram rapidamente: Os cookies do Twitter/X expiram com mais frequência do que outros sites. Se o gallery-dl parar de baixar, basta executar novamente com --cookies-from-browser chrome para atualizar a sessão.

Comandos

Comandos gallery-dl Twitter/X para Windows

Baixar todas as mídias da aba de mídia de um usuário
gallery-dl --cookies-from-browser chrome "https://x.com/username/media"
Baixar mídia de um único tweet
gallery-dl --cookies-from-browser chrome "https://x.com/username/status/TWEET_ID"
Baixar linha do tempo do usuário
gallery-dl --cookies-from-browser chrome "https://x.com/username"
Baixar aba de curtidas
gallery-dl --cookies-from-browser chrome "https://x.com/username/likes"
Baixar com pasta de saída personalizada
gallery-dl --cookies-from-browser chrome -d "C:\Imagens\Twitter" "https://x.com/username/media"
Baixar com cookies.txt (para tarefas agendadas)
gallery-dl --cookies "C:\Tools\gallery-dl\twitter-cookies.txt" "https://x.com/username/media"

Configuração para Twitter/X

Configuração recomendada do Twitter/X para gallery-dl no Windows

config.json — seção Twitter
{ "extractor": { "twitter": { "cookies": "C:/Tools/gallery-dl/twitter-cookies.txt", "videos": true, "retweets": false, "quoted": true, "filename": "{author[name]}_{tweet_id}_{num}.{extension}", "sleep-request": 1.0 } } }
videos: true
Baixar arquivos de vídeo também
retweets: false
Pular retweets — apenas postagens originais
quoted: true
Incluir mídia de tweets citados
sleep-request: 1.0
Pausa entre requisições para evitar rate limits

Perguntas Frequentes

gallery-dl Twitter/X — perguntas frequentes

O gallery-dl funciona com Twitter/X em 2026?
Sim. O extrator do gallery-dl para Twitter/X é mantido ativamente. Desde as restrições de API do Twitter em 2023, login via cookies é obrigatório. Mantenha atualizado com gallery-dl -U.
gallery-dl Twitter diz "autenticação necessária" com cookies
Os cookies do Twitter/X expiram frequentemente. Faça logout e login novamente em x.com no navegador, feche completamente o navegador, depois execute novamente com --cookies-from-browser chrome.
gallery-dl baixa imagens do Twitter em baixa resolução
Por padrão, gallery-dl adiciona ?format=jpg&name=orig para solicitar resolução original. Se ainda estiver baixando em baixa resolução, atualize com gallery-dl -U.
Como baixo apenas imagens e pulo vídeos?
Adicione um filtro de imagem: gallery-dl --image-filter "extension in ('jpg','png','webp','gif')" --cookies-from-browser chrome "URL". Ou defina "videos": false na seção Twitter do config.json.
gallery-dl pode baixar de contas privadas/protegidas do Twitter?
Sim, se você seguir a conta e estiver logado. Passe seus cookies do navegador — gallery-dl acessará a linha do tempo usando sua sessão autenticada.
