Instalação passo a passo

Como instalar gallery-dl no Windows em 5 passos

Funciona no Windows 11 e Windows 10 (64 bits). Sem assistente de instalação, sem direitos de administrador.

Passo 01

Baixar gallery-dl.exe

Vá para a página de download e salve gallery-dl.exe da versão oficial do GitHub.

Passo 02

Salvar em pasta dedicada

Mova o arquivo para um local permanente, ex: C:\Tools\gallery-dl\. Evite Downloads — arquivos lá podem ser apagados automaticamente.

Passo 03

Abrir PowerShell nessa pasta

No Explorer, clique na barra de endereços, digite powershell e pressione Enter. O PowerShell abre diretamente na pasta correta.

Passo 04

Verificar se funciona

Execute gallery-dl --version. Um número de versão como 1.28.x deve aparecer. Se sim, está funcionando.

Passo 05

Atualizar para a versão mais recente

Execute gallery-dl -U. Baixa e substitui automaticamente pela versão mais recente. Faça isso regularmente.

Opcional

Configurar arquivo de configuração

Crie um gallery-dl.conf para definir pastas de saída padrão, modelos de nome de arquivo e cookies. Guia de configuração →

Opcional mas recomendado

Como adicionar gallery-dl ao PATH no Windows

Com gallery-dl no PATH você pode executar gallery-dl de qualquer pasta no PowerShell ou CMD.

Método gráfico (recomendado)

  1. Pressione Win e pesquise Variáveis de Ambiente
  2. Clique em Editar variáveis de ambiente do sistema
  3. Em Variáveis do usuário, selecione Path → Editar
  4. Clique em Novo e cole o caminho da pasta, ex: C:\Tools\gallery-dl
  5. Clique OK em todos os diálogos e reinicie o terminal

Método PowerShell

Execute uma vez no PowerShell — define PATH apenas para seu usuário, sem direitos de admin:

setx PATH "%PATH%;C:\Tools\gallery-dl"

Feche e reabra o PowerShell. Teste com gallery-dl --version de qualquer diretório.

Nota: Se aparecer "gallery-dl não é reconhecido" após adicionar ao PATH, feche todas as janelas do terminal e reabra — mudanças de PATH só entram em vigor em novas sessões.

Problemas comuns

Solucionar problemas de instalação do gallery-dl no Windows

Aviso do Windows SmartScreen ao executar gallery-dl.exe
Clique em Mais informações → Executar mesmo assim. O SmartScreen sinaliza novos executáveis por padrão. gallery-dl.exe é a versão oficial de código aberto do GitHub. Alternativamente: clique com o botão direito → Propriedades → marque Desbloquear na parte inferior → OK.
"gallery-dl não é reconhecido" no PowerShell
Duas opções: execute gallery-dl diretamente de sua pasta, ou adicione a pasta ao PATH (veja seção acima). Após adicionar ao PATH, feche todas as janelas do terminal e reabra.
"Acesso negado" ou arquivo bloqueado pelo antivírus
Clique com o botão direito em gallery-dl.exe → Propriedades → marque Desbloquear na parte inferior se visível → OK. Se o antivírus bloquear, adicione uma exceção — é um falso positivo para portáteis de código aberto.
gallery-dl não baixa nada / fecha imediatamente
O site provavelmente requer login. Execute gallery-dl --cookies-from-browser chrome "URL". Veja o guia de cookies para detalhes completos.
