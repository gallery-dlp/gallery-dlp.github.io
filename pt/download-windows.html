Início rápido

Após baixar gallery-dl.exe

Quatro passos do download até sua primeira galeria de imagens.

Passo 01

Salvar em uma pasta

Coloque gallery-dl.exe em uma pasta permanente como C:\Tools\gallery-dl\ — evite Downloads que pode ser limpa automaticamente.

Passo 02

Se o navegador alertar

Clique em Manter → Manter mesmo assim. Se aparecer SmartScreen: Mais informações → Executar mesmo assim.

Passo 03

Abrir PowerShell na pasta

Clique na barra de endereços do Explorer, digite powershell e pressione Enter. O PowerShell abre diretamente na pasta certa.

Passo 04

Verificar & executar

Digite gallery-dl --version para confirmar que funciona, depois cole qualquer URL para começar a baixar.

Perguntas Frequentes

FAQ sobre o download

Preciso de direitos de administrador para executar gallery-dl.exe?
Não. gallery-dl.exe é portátil — sem instalação necessária. Direitos de administrador só são necessários se quiser adicionar a pasta ao PATH do sistema para todos os usuários.
Preciso do Python instalado para usar o gallery-dl no Windows?
Não. A versão gallery-dl.exe inclui um runtime Python internamente. Basta baixar o .exe e executar — sem instalação do Python necessária.
Como baixo de uma conta privada do Instagram ou Twitter?
Passe cookies do navegador: gallery-dl --cookies-from-browser chrome "URL". Veja o guia completo de cookies e login.
Como mantenho o gallery-dl atualizado?
Abra o PowerShell na pasta do gallery-dl.exe e execute gallery-dl -U. Atualiza automaticamente para a versão mais recente.
