Início rápido

Como instalar o gallery-dl no Windows em 4 passos

Sem direitos de administrador. Funciona em qualquer PC com Windows 11 ou Windows 10.

Passo 01

Baixar gallery-dl.exe

Salve em qualquer pasta, ex: C:\Tools\gallery-dl\ — sem instalação, totalmente portátil.

Passo 02

Abrir PowerShell nessa pasta

Clique na barra de endereços do Explorer, digite powershell e pressione Enter.

Passo 03

Verificar a instalação

Execute gallery-dl --version. Se aparecer um número de versão, está pronto.

Passo 04

Baixar sua primeira galeria

Cole qualquer URL suportada e pressione Enter. O gallery-dl salva imagens em resolução original.

Comandos gallery-dl para Windows

Comandos gallery-dl mais úteis — copie e cole

Execute diretamente no PowerShell ou CMD. Substitua a URL pelo seu link de destino.

Baixar todas as imagens de uma URL
gallery-dl "https://www.instagram.com/username/"
Baixar com cookies do navegador (para sites que exigem login)
gallery-dl --cookies-from-browser chrome "URL"
Salvar em pasta personalizada com nomes de arquivo originais
gallery-dl -d "C:\Imagens\%(category)s\%(uploader)s" "URL"
Baixar apenas imagens (ignorar vídeos)
gallery-dl --image-filter "extension in ('jpg','png','webp')" "URL"

Perguntas Frequentes

Perguntas frequentes sobre gallery-dl para Windows

O gallery-dl é seguro no Windows?
Sim. O gallery-dl é totalmente código aberto e hospedado no GitHub. Cada linha de código é auditada publicamente. Sem software adicional, sem telemetria, sem serviços em segundo plano.
O gallery-dl funciona no Windows 11?
Sim. gallery-dl.exe funciona no Windows 11 e Windows 10 (64 bits) sem instalação. Baixe o .exe, abra o PowerShell e execute o primeiro comando.
Quais sites o gallery-dl suporta?
O gallery-dl suporta 200+ sites incluindo Instagram, Twitter/X, Pixiv, DeviantArt, Reddit, Tumblr, Danbooru, Flickr, Artstation, Patreon e muito mais. Veja a lista completa no GitHub.
Preciso do Python para usar o gallery-dl no Windows?
Não. A versão gallery-dl.exe inclui um runtime Python integrado. Basta baixar o .exe e executar — sem instalação do Python necessária.
Como atualizo o gallery-dl no Windows?
Abra o PowerShell na pasta do gallery-dl e execute gallery-dl -U. Atualiza automaticamente para a versão mais recente.
Por que o gallery-dl diz "login necessário" ou "403 Forbidden"?
Muitos sites exigem autenticação para conteúdo privado. Use --cookies-from-browser chrome para passar sua sessão do navegador. Veja o guia completo de cookies.
