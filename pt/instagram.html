gallery-dl Instagram no Windows — Baixar Fotos, Reels & Stories

gallery-dl baixa perfis completos, postagens, Reels, Stories e destaques do Instagram em resolução original. Login via cookies do navegador é obrigatório desde as restrições de acesso público do Instagram em 2023.

⬇ Baixar gallery-dl.exe Guia de configuração de cookies →
Atualização 2026 — login obrigatório: Desde meados de 2023, o Instagram requer login para quase todo o conteúdo. Passe seus cookies do navegador com --cookies-from-browser chrome. Certifique-se de estar logado no Instagram no seu navegador primeiro.

Comandos

Comandos gallery-dl Instagram para Windows

Todos os comandos executados no PowerShell da pasta contendo gallery-dl.exe.

Baixar todas as postagens de um perfil público ou seguido
gallery-dl --cookies-from-browser chrome "https://www.instagram.com/username/"
Baixar uma única postagem
gallery-dl --cookies-from-browser chrome "https://www.instagram.com/p/POST_ID/"
Baixar Stories (login obrigatório)
gallery-dl --cookies-from-browser chrome "https://www.instagram.com/stories/username/"
Baixar Destaques
gallery-dl --cookies-from-browser chrome "https://www.instagram.com/highlights/HIGHLIGHT_ID/"
Salvar em pasta específica com subpasta de nome de usuário
gallery-dl --cookies-from-browser chrome -d "C:\Imagens\Instagram" "https://www.instagram.com/username/"
Baixar com arquivo cookies.txt salvo (para automação)
gallery-dl --cookies "C:\Tools\gallery-dl\instagram-cookies.txt" "https://www.instagram.com/username/"

Configuração para Instagram

Definir cookies do Instagram permanentemente na configuração do gallery-dl

config.json — seção Instagram
{ "extractor": { "base-directory": "C:/Imagens/", "instagram": { "cookies": "C:/Tools/gallery-dl/instagram-cookies.txt", "sleep-request": 3.0, "videos": true, "filename": "{username}_{shortcode}_{num}.{extension}" } } }

sleep-request: 3.0 — reduz as requisições para evitar rate limiting do Instagram. Recomendado para downloads de perfis grandes.

Perguntas Frequentes

gallery-dl Instagram — perguntas frequentes

O gallery-dl ainda funciona com o Instagram em 2026?
Sim, mas login é obrigatório. O Instagram restringiu o acesso público em 2023. O extrator do Instagram no gallery-dl é mantido ativamente. Mantenha atualizado com gallery-dl -U.
gallery-dl diz "Instagram requer login" mesmo com cookies
Seus cookies podem ter expirado. Faça logout e login novamente no Instagram no navegador, depois execute novamente com --cookies-from-browser chrome. Certifique-se de que o navegador está completamente fechado antes de executar gallery-dl no Windows.
gallery-dl Instagram está muito lento — como acelerar?
O Instagram limita scrapers agressivos. Não reduza sleep-request abaixo de 2 segundos ou sua conta pode ser temporariamente restringida. Use --download-archive archive.txt para pular arquivos já baixados.
gallery-dl pode baixar Reels do Instagram?
Sim. Reels são baixados como arquivos de vídeo .mp4. Use a URL do perfil para todos os Reels de um usuário, ou a URL direta do Reel para um único vídeo. Login via cookies é obrigatório.
Como pulo arquivos já baixados?
Use a flag de arquivo de download: gallery-dl --download-archive archive.txt --cookies-from-browser chrome "URL". O gallery-dl registra cada ID de arquivo baixado no arquivo e pula nas execuções futuras.
