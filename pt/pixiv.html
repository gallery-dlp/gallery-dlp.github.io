gallery-dl Pixiv no Windows — Baixar Obras de Arte, Galerias & Favoritos

gallery-dl baixa galerias completas de artistas do Pixiv, seus favoritos, buscas por tag e animações ugoira. Uma conta gratuita do Pixiv e login por cookie são necessários para acesso completo.

⬇ Baixar gallery-dl.exe Guia de configuração de cookies →

Comandos

Comandos gallery-dl Pixiv para Windows

Baixar galeria completa de um artista
gallery-dl --cookies-from-browser chrome "https://www.pixiv.net/users/USER_ID"
Baixar uma obra de arte individual (todas as páginas)
gallery-dl --cookies-from-browser chrome "https://www.pixiv.net/artworks/ARTWORK_ID"
Baixar seus favoritos
gallery-dl --cookies-from-browser chrome "https://www.pixiv.net/users/SEU_USER_ID/bookmarks/artworks"
Baixar obras de arte por tag
gallery-dl --cookies-from-browser chrome "https://www.pixiv.net/tags/NOME_TAG/artworks"
Baixar feed de seguindo (novas obras dos artistas que você segue)
gallery-dl --cookies-from-browser chrome "https://www.pixiv.net/bookmark_new_illust.php"
Salvar em pasta personalizada com subpasta de nome do artista
gallery-dl --cookies-from-browser chrome -d "C:\Imagens\Pixiv" "https://www.pixiv.net/users/USER_ID"

Animações Ugoira

Como baixar ugoira do Pixiv com gallery-dl no Windows

Ugoira são as animações baseadas em frames do Pixiv. gallery-dl pode baixar o ZIP bruto ou converter para GIF/WebM com FFmpeg.

Padrão — baixar como ZIP

gallery-dl baixa ugoira como arquivo ZIP contendo frames individuais. Sem FFmpeg necessário.

gallery-dl --cookies-from-browser chrome "https://www.pixiv.net/artworks/UGOIRA_ID"

Converter para GIF ou WebM (requer FFmpeg)

Adicione ao config.json para converter ugoira automaticamente:

"pixiv": { "ugoira": true, "ffmpeg-args": ["-vf", "scale=600:-1"], "ffmpeg-output": "webm" }

Perguntas Frequentes

gallery-dl Pixiv — perguntas frequentes

gallery-dl requer uma conta Pixiv?
Sim. Uma conta gratuita do Pixiv é necessária. Sem login, gallery-dl só pode acessar um pequeno subconjunto de obras públicas. Faça login em pixiv.net no seu navegador, depois use --cookies-from-browser chrome.
gallery-dl pula obras R-18 do Pixiv — como corrigir?
Conteúdo R-18 requer uma conta Pixiv com conteúdo adulto habilitado nas configurações da conta. Faça login em pixiv.net, ative R-18 nas configurações, depois passe cookies para gallery-dl.
Como encontro o ID de usuário Pixiv de um artista?
Abra o perfil do artista no Pixiv no seu navegador. A URL será como pixiv.net/users/12345678 — o número é o ID de usuário. Você pode usar a URL completa diretamente com gallery-dl.
gallery-dl baixa ugoira como ZIP em vez de GIF — como converter?
Coloque FFmpeg na pasta do gallery-dl (C:\Tools\gallery-dl\ffmpeg.exe) e defina "ugoira": true e "ffmpeg-output": "gif" na seção Pixiv do config.json. gallery-dl então converterá ugoira automaticamente para GIF ou WebM.
Como baixo apenas ilustrações (sem manga/ugoira)?
Use um filtro de imagem: gallery-dl --image-filter "type == 'illust'" --cookies-from-browser chrome "URL". Os tipos são: illust, manga, ugoira.
