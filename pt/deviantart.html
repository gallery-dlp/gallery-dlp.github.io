gallery-dl DeviantArt no Windows — Baixar Galerias & Favoritos

gallery-dl baixa galerias completas de usuários, favoritos, coleções e deviations individuais do DeviantArt em resolução original. Galerias públicas funcionam sem login — conteúdo maduro requer cookies.

⬇ Baixar gallery-dl.exe Guia de configuração de cookies →

Comandos

Comandos gallery-dl DeviantArt para Windows

Execute no PowerShell da pasta contendo gallery-dl.exe. Galerias públicas funcionam sem cookies.

Baixar galeria completa (pública — sem login necessário)
gallery-dl "https://www.deviantart.com/username/gallery/"
Baixar galeria com login (para conteúdo maduro)
gallery-dl --cookies-from-browser chrome "https://www.deviantart.com/username/gallery/"
Baixar favoritos
gallery-dl --cookies-from-browser chrome "https://www.deviantart.com/username/favourites/"
Baixar pasta de coleção específica
gallery-dl --cookies-from-browser chrome "https://www.deviantart.com/username/gallery/FOLDER_ID/nome-pasta"
Baixar uma deviation individual
gallery-dl "https://www.deviantart.com/username/art/nome-obra-123456789"
Salvar em pasta personalizada
gallery-dl -d "C:\Imagens\DeviantArt" "https://www.deviantart.com/username/gallery/"

Configuração para DeviantArt

Configuração recomendada do DeviantArt para gallery-dl no Windows

config.json — seção DeviantArt
{ "extractor": { "deviantart": { "cookies": "C:/Tools/gallery-dl/deviantart-cookies.txt", "original": true, "metadata": true, "filename": "{author}_{index}_{title}.{extension}", "sleep-request": 1.0 } } }
original: true
Baixar arquivo original enviado (sem redimensionamento)
metadata: true
Salvar metadados da deviation como JSON

Perguntas Frequentes

gallery-dl DeviantArt — perguntas frequentes

gallery-dl baixa DeviantArt em resolução original?
Sim, quando "original": true está definido na configuração (ou com sessão logada). Sem isso, gallery-dl pode baixar a versão de exibição web. Algumas deviations mais antigas só têm versão web-resolution disponível.
gallery-dl DeviantArt diz "filtro de conteúdo" ou pula deviations
Conteúdo maduro é filtrado para visitantes não logados. Faça login no DeviantArt no navegador e passe cookies: --cookies-from-browser chrome. Certifique-se de que conteúdo maduro está habilitado nas configurações da conta DeviantArt.
gallery-dl está baixando DeviantArt muito lentamente
DeviantArt tem limites de taxa. O padrão sleep-request: 1.0 geralmente é suficiente. Se receber erros 429, aumente para 2.0 ou 3.0. Não execute múltiplas instâncias simultâneas do gallery-dl no DeviantArt.
Como baixo apenas uma pasta específica de uma galeria do DeviantArt?
Navegue até a pasta no DeviantArt, copie a URL (contém o ID da pasta) e passe diretamente para gallery-dl. O formato da URL é deviantart.com/username/gallery/FOLDER_ID/nome-pasta.
